# 📋 内部文件收集与管理工具（截图收集系统）

## 项目简介

这是一个用于收集和管理员工转发公众号/视频号内容截图的内网工具系统。系统实现了截图文件的智能化收集、规范化存储、便捷式上传和实时化统计。

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite + Element Plus + Pinia + Vue Router
- **后端**: Go + Gin + GORM + PosgreSQL

### 前置要求

- Go 1.25.4+
- Node.js 18+
- pnpm
- MySQL 5.7+

## 功能模块

- ✅ 部门管理
- ✅ 人员管理
- ✅ 转发任务管理
- ✅ 文件上传（支持剪贴板、拖拽、选择文件）
- ✅ 实时统计看板
- ✅ 文件导出（ZIP压缩）
- ✅ 报表导出（Excel）
